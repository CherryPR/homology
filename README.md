　　西海峰林独属于虹猫的院子里，种满了桃树，当年魔教放火烧山，半个山头的花草林木被毁得一干二净，此处却因地势高峻隐蔽而幸免于难。
　　蓝兔第一次来时恰好是春天。
　　几回红绽雨添肥，草木发新，花枝迤逦。
　　其实论盛景，是比不上玉蟾宫那片十里桃林的，可这儿的桃树生在悬山中，荒得直往高里长，最高的要数院中那棵，稠密的枝杈错落伸展，孔雀尾似的散开，几乎盖住了大半个院子——这是蓝兔见过的最大的桃树，直到后来他们天南地北游历四方，见识过许多千奇百怪的古树异花后，这印象依然如此。
　　“我爹说自他记事起，这棵树就这么高这么大了，所以谁也不清楚究竟有多少年头。”
　　“小时候，我每日都会在树下练上两个时辰的剑法，然后在这树上看半个时辰的书。”
　　再之后，便趁爹娘不注意，折了几根桃枝挡脸，悄悄猫着腰蹿下树来，骑上麒麟溜到后山去玩，等到天快黑的时候再越墙而入。
　　这样，一天也就消磨过去了。
　　虹猫记得，从门口走到院中这棵桃树下，是三十七步，他还记得，有一次在外面直耍到月上中天才尽兴而归，像往常一样翻墙爬进来，殊不知爹爹正守在墙根等他，直接踩着爹爹的肩膀跳进了院子。
　　结果自然是被追着满院子打。
　　
　　
　　繁叶如海，旁枝斜伸如泊在海边的船，载着两个人悠悠荡荡。
　　触目可及峰林，伸手可摘山花，蓝兔一抬头就看见被纵横的枝条切割成碎块的蓝天，秾芳一朵朵缀乱云霞，流连在枝桠之末，如烟似锦，却兜不住从缝隙中漏下来的日光，透进亮与暖，熏得人也懒意昏昏。
　　不起眼的树洞里，藏着虹猫曾经看过的书，封皮光秃泛黄、页脚翘起来的，可见旧时经常翻动，多半是话本、戏文之类不那么正经的，而那些崭新的仿佛还能闻见墨味的，无一例外皆是些儒经、道经、佛经，一读就头晕得厉害。
　　她掏出一本，朝靠着树干的人扬了扬，笑吟吟道：“只怕那半个时辰全用来偷看这些不着调的闲书了。”
　　“偷得浮生半日闲。”虹猫从她手里接过，直接扣在了脸上，将清隽而又懒倦的眉眼遮住，“话本子可比经书有意思多了。”
　　这倒也是——蓝兔在心里点头附和，他喜欢话本，她也喜欢话本，尽管他俩的喜欢是不同的，他钟爱仗剑江湖、快意恩仇，可她偏爱山海志怪、仙灵妖奇。
　　树洞的最里面有一本小册子，似与别物不同，蓝兔眼尖地瞧见，正打算抽出来，旁边的人却先她一步将其抢到手中。
　　“什么东西？给我看看！”
　　即使两人都坐着，虹猫还是肉眼可见比她高出小半截，仗着身量优势，一脸淡定地将手拿远举高，扔出一句：“不给。”
　　越是东遮西掩，越引得人想一探究竟，蓝兔又犯了好奇心过剩的毛病，扯着他的袖子便往人身上攀，却被拦腰搂住。
　　趁她愣神之际，虹猫手上微一施力, 就把她拉到了自己腿上，顺势将那本册子塞进她怀里，埋首在她颈侧低语：“想看就看吧。”
　　一如往常舒朗的声音，却无端听出了些别样的、不外透的眷意，附着在耳骨上，好像被绒花芽尖最柔的碎絮擦过，蕴出一点让人耳根酸软的悸动。
　　手里的东西突然变成了烫手山芋——这让人怎么看啊，她的手指搭在起褶的书封上，迟迟没有动作。
　　“方才不是闹着要看，这会儿怎么消停了。”
　　“你起开一点，我没法看了……”
　　“就这样看。”
　　虹猫双手又从后面揽回来，圈得越发紧了，面上仍是气定神闲，好心替她翻开书页，“从后往前看。”
　　眼见着一身罗裙又要被蹭得乱糟糟，蓝兔手肘向后撞了一下，以示警告，尔后垂下脑袋，视线落在册子中央正立于树下练剑的一个白衣小少年身上，约莫六七岁的模样，面目青涩，身形还未完全长开，一招一式却走得有模有样，行云流水间不知拂落多少飞花。
　　“这是你小时候？”
　　“嗯。”他挨着她肩膀的下巴动了动，“我娘从前最喜爱作画。”
　　蓝兔往前翻了翻，那画册上，有他念书的样子，有他嬉戏的样子，有他皱着眉头、眼泪涟涟的样子，有他兴高采烈、眉飞色舞的样子，也有什么都不做只是躺在树上安静睡觉的样子。
　　诸般神态，皆汇于一册，是她从未见过的年少模样。
　　指尖轻轻碰了碰画中人有些圆嘟嘟的脸颊，蓝兔笑着打趣道：“你小时候可比现在可爱多了。”
　　虹猫只是一言不发地环着她，难得没有反驳。
　　又翻到某一页，还未看清楚忽被盖住，甚至试图扯着抢回去，幸而她眼疾手快，一把打落那只爪子，“急什么？我还没看完呢。”
　　“剩下的没什么可看的了。”
　　枝叶摇曳，浮光下彻，将虹猫脸上微不可见的一丝古怪照得十分明晰。
　　蓝兔才不管他，重新打开，待看清那册上的画面，愣了一瞬，忽而笑出了声。
　　怪不得死活不肯叫她看，原是个年娃娃似的小人，红肚兜、光屁股，滚在地上一身泥，怀里还抱着条滑不溜秋的大鲤鱼。
　　大鱼扑腾得正欢，小娃娃哭得也欢。
　　“没想到……”她趴在他肩头，乐得弯了眉眼，话也断断续续，“没想到我们虹猫少侠还有这么、这么没出息的一面……”
　　“有那么好笑？”
　　看着面无表情，心里说不定怎么臊得慌呢，平时老被他调侃，今日总算抓了他一个小尾巴，蓝兔又好玩似的捏住他的耳朵尖揪了两下，直到滑溜溜的一团软肉被她蹂躏得通红，才心满意足道：“害什么羞呀，你那光屁股蛋我都看了好几回了，也不差这个小的。”
　　话音刚落，腰间那根系带突然被他拽住轻扯，蓝兔大惊失色，慌忙伸臂去挡，却被施施然捉住，动弹不得。
　　“你做什么！”
　　她今日只穿了件鹅黄素纱抹胸，外面罩着同色单罗衫，拉扯间稍稍松散，最惹眼的莫过于肩头露出的一点莹润，光扫过来，亮了一下，又沉入阴影里。
　　虹猫原并未有旁念，不过想逗她一逗，只是此时不算平稳的呼吸出卖了他。
　　“我要做些什么，你还不清楚？”
　　含笑的声音一路从脖子逡巡至耳后，明明是清雅初春，却闷得蓝兔忍不住想躲，还未来得及就被衔住了耳垂，不轻不重地碾了下，原本只有一点薄粉的耳朵，迅速晕染成明脆的绯色，竟真的像兔子一样抖了一抖。　　
　　“自然是礼尚往来，也瞧瞧你的光屁股蛋。”
　　
　　
　　见他把自己的话一字不差地还了回来，蓝兔霎时间大窘，啐道：“青天白日的，也好意思！”
　　“青天白日怎么了，这种事还要分时辰？”
　　蓝兔简直被这人的厚脸皮惊到了，怎么什么话都说得这般理直气壮，想踢他、推他，可腕骨还硌在他掌心里，刻意用指腹摩挲小小的凸起，霜肌沁凉，独被他磨出红痕的地方一阵阵发烫。
　　那双漂亮的眼又离得这样近，不知为什么连压下来的目光也如此灼灼，仿佛藏了钩子，被火燎过，还吊着余温，从她脖颈处汩汩跳动的脉搏，从她月牙一样纤巧的锁骨，从她春衫之下若隐若现的隆起，每带过一寸都如有实质，穿透衣物，潜入皮肉，在身体里勾起细小又暧昧的热潮。
　　一句反驳的话也讲不出来。
　　直至林间树叶的沙沙呢喃，被漫步的风捎送到耳畔，蓝兔才倏然醒觉，此处虽不算席地，可跟幕天也没什么区别，到底还是存了几分羞赧心，红着脸吐出一句：“到树下去……”
　　连腕子没怎么用力的挣扎都成了催人魂魄的欲拒还迎。
　　锢着她的手忽然松了力道，蓝兔不由抬起眼来，却见虹猫半歪着脑袋倚在树上，下颌微微扬起一点，依旧是一副似笑非笑的模样。
　　“你确定？树下可都是野草丛，扎屁股得很。”
　　又在故意曲解她的意思，可恶得叫人只想咬上一口，于是嘴巴先于理智扑了上去，倒像主动将自己送到他嘴边似的，一个吻落下来，无从躲避。
　　
　　
　　奇妙的清香蓦地渡入口中，蓝兔朦朦胧胧地猜，这人是不是偷吃了桃子啊，又或者是杏子、李子、梅子，不对，都不对，李子微涩，杏甘中带苦，梅子留酸软齿牙，猜不准，也没有关系，这味道如何沁人心脾，这一吻便如何绵密悠长，足够她将所有相似的气味回忆个遍。
　　久闻不厌，甚至足以达到上瘾的地步，又令她生出了一种嗅觉味觉都被网住的错觉，抵在他胸口的手免不了要小小地推拒一番，结果便是这点微弱的抵抗被人犀利地察觉，捧着她后脑勺的手拔了头上拢发的簪，青丝如瀑，泻了满背。
　　少年修长的手指穿梭在她发间，另一只手覆在细腰上将她一把拘住，按着她全然靠向自己，启开檀口便趁机更进了一步去。
　　黏黏糊糊中带着一丝不满，似乎在埋怨她给的不够多。
　　但好在这个时候她总是格外乖，毫不鄙吝地松软了牙关任其攻城掠地。
　　那小舌尖无论如何藏都能轻易被他找到，继而紧密又灵活地卷缠在一起，像蜜蜂穿荆越棘，落歇蔷薇，咬着一段初生的花苞，一点一点汲走瓣叶上的雨露，直要将里面的花汁都攫尽才肯罢休。
　　树上不知名的小虫叫得噪耳，更噪耳的是唇齿交融间搅起的沥沥水声，与春风的私语撞在一起，似乎将叶影也荡得斑驳。
　　明明是要咬他来着，怎么反过来又被他咬了。
　　更过分的是，他亲她，她也回应，怎么每次都亲不过他。
　　蓝兔眼色昏昏，沉浸在这一记密不透风的长吻，仿佛气息吞吐都被左右，不多久便软成了一汪水，只剩下蒲苇般柔韧的双臂从水里探身而出，悄无声息地攀上他的肩背，越贴越紧，越贴越密。
　　这对一向擅长掌控的虹猫来说，何尝不是折磨。
　　纱罗薄若雾谷，哪怕只是轻轻一动，隐藏在下面的温腻细滑，也能毫无保留地传导给他。
　　两团莹软从他胸前无措地蹭过，就像兔子撞进青草地的怀抱，偷偷撒野，连心跳都无数倍地放大，一波波敲击着耳膜，充斥在脑海，鼓噪着占有，渴望着更亲密的接触。
　　他却并不着急，反倒享受起这快活的折磨。
　　从前四处奔波时，瞧着她哪哪都伶仃一把骨，如今倒养出了些丰润来，俏生生、挺茁茁，如同夏令河池盛绽的玉芙蓉，又软腴得像是刚出锅的鲜嫩豆腐，不过随意一压，便失了形状，往中间挤出道艳靡的沟壑，在齐胸的衣襟下似水一样晃溢开来。
　　那颗曾在夜里吮弄过不知多少次的小荷尖，也因为这放荡的厮磨悄悄立了起来，顶着层层软布，倔强地用最脆弱敏感的顶端对抗着他坚硬结实的胸膛。
　　“嗯…….”蓝兔在无意识中发出含糊的轻吟，被他的舌头挡了一半，漏出来的声音软糯得要滴蜜。
　　哪里是平时那只清凌凌的兔子，更像是被舔了后颈的奶猫——虹猫心中多了丝了然的愉悦，她素性触痒不禁，此刻一定连脊骨缝里都像有蚂蚁在爬。
　　如他所想，两人外衫都没脱，就这样贴着身子歪缠，挺翘的乳尖与衣服上的密绣花纹搽来捻去，有种绵麻的痒从细细的小眼里发酵蔓延，如野草疯长，一下就窜满了全身，连带着最隐秘的地方也渐渐漾开波澜。
　　裙衫太轻薄了，蓝兔有点担心会不会把他的衣服也波及。
　　但显然失礼的人不止她一个——他身下的那东西不知什么时候已奋跃起来，硬硬地抵在她的臀缝，一跳一跳，伴随着两道越吻越乱的呼吸，没有章法地轻磨，磨得她好酸，炽烫得她面颊烧起来，是日中清影也掩不住的醉酡。
　　原本安放于腰间的手也摸索着往上探，拢住了她的一边乳，在指尖肆无忌惮地搓揉捏弄。
　　蓝兔圈在他脖子上的胳膊不知道是该推开还是该搂紧，恰如在他隔着衣衫抚摸自己时，不知道是想让他轻一点，还是重一点。
　　每一处被他触碰过的地方都浮起红晕，没被触碰过的反而起了焦躁，像放进逐渐变热的温水里慢煮，煎熬得里外熟透还浑然不知。
　　痛快又不痛快。
　　青天白日也好，幕空席地也好，好像都没那么重要了，阳光点在睫上炫起的斑斓将视野模糊，只有唇舌的热和软是真实的，只有腿心的湿与乱是真实的，蓝兔屈起膝盖暗暗往前，一下下勾着他的胯骨。
　　怎么痒，她就怎么蹭，像是不熟练的引诱。
　　未着鞋履的脚尖，从长裙底下露出一点来，通透白润，似笼了莹莹的晕，悬在空中一荡一荡，划出惑人的弧度。
　　虹猫喘息重了一瞬，按住她的膝弯，封锁住她所有的动作，“你真是……”
　　真是个妖精。
　　但也妖得可爱就是了。
　　

　　他的腕骨痒得有些发疼，虚虚撩开裙裾，游蛇一样不安分地行走在她小腿上，时上时下描绘着玲珑的曲线，收着力气，也抑着喘息，仿佛在把玩一樽如脂似玉的釉瓷器，又谨记要轻拿轻放的原则，生怕打碎了碰坏了。
　　只是蓝兔一向知晓，愈是小心体贴，愈是不怀好意。
　　果然他的手一个用力，便将里面的亵裤撕了个干净。
　　裂帛声叫她从万千缠绵中剥出一丝清醒来，可舌头被叼得厉害，实在分不出空去斥责这得寸进尺的采花贼，只在心里将此人骂了个狗血喷头。
　　怎么就不能斯文一点。
　　似是听到了她的控诉，唇舌的遏制终于消失了，蓝兔无力地伏在他身上平复呼吸，日光也从花隙间无力地打下来，颓唐着漫开一圈金黄的晕环，遮住了迷蒙的视线。
　　两人都短暂汲取着新鲜空气，她是眼角眉梢都红透了的轻喘，他则是留恋齿颊生香的过分美好而炙烈粗喘。
　　“往后少吃些糖。”
　　她嘴里的味道太甜，不是蜜枣一样的齁甜，而是像在井水中湃过的西瓜，清甜中沾染了凉意，抿一口，便再也舍离不得。
　　蓝兔觉得他这话说得奇怪，又认真地想了想，除了春日的拔丝雪梨、夏日的冰镇水蜜桃、秋日的醪糟汤圆，还有冬日的烤红薯，她也不怎么爱吃甜的。
　　反倒应该是他，少吃些乱七八糟的野果子，她都尝不出来是什么味道了。
　　她思绪游离地飘着，可裙下的手未曾离开，满心的不安还在滋长。
　　“又走神。”
　　许是不满她的心不在焉，长指从后往前滑，悄然滑进了桃子一样的两瓣臀肉中，是蜻蜓点水般的肤浅力道，只不经意地一挑，便涟漪自生，惊起含着媚意的吟咛。
